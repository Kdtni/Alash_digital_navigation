renderComparison() - основа:
Эта функция делает почти всё:
    Получает выбранные университеты
    Очищает таблицу
    Если ничего не выбрано то выводит сообщение
    Заполняет заголовки
    Добавляет строки критериев
    Генерирует карточки "Победитель по критерию"



updateTableHeaders():
function updateTableHeaders(selectedUnis) { ... }

Заполняет верх таблицы:
    логотип
    название
    город
    рейтинг

Если вуз не выбран то пишет "Не выбран"



generateComparisonResults():
Создаёт блоки:
    лучшая стоимость
    лучший рейтинг
    лучшие перспективы трудоустройства
    рекомендация

Каждое сравнение строится на отдельных функциях.



compareByCost() - сравнение стоимости:
    Извлекает цифры из строки "от 600,000 тг/год"
    Находит минимальную
    Возвращает вывод + победителя

compareByRating() - Работает аналогично:
universitiesData[id].rating
Находит максимальный — выводит победителя.

compareByEmployment() Ищет процент:
"85%" → 85
Находит максимум.



generateRecommendation()
Алгоритм простой:
    Лучший рейтинг = лучший университет
    Возвращает его как рекомендацию



Обработчики событий:
compareBtn.addEventListener("click", renderComparison);

Кнопка сравнить - вызывает обновление
Кнопка сброса - очищает выбор
Dropdown меню - автоматически обновляет таблицу



Инициализация (DOMContentLoaded):
renderComparison();
делает первую отрисовку.
